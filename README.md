### 1.2 什么是单点登录

单点登录就是在一个多系统共存的环境下，用户在一次登录后，就不用在其他系统
中登录，也就是用户的一次登录能得到其他所有系统的信任

就比如在支付宝登录了，在淘宝，天猫也可以登录
